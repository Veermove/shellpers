# DNF

## history
```
dnf history
dnf history last
dnf history last-3
dnf history 226
```

## transaction info
```
dnf hisory info last
dnf history info 226
```

## repo list
```
dnf repolist --all
```

## module list
```
dnf module list
```

## group list
```
dnf grouplist
```

## query packages in repo
```
dnf --disablerepo="*" --enablerepo="REPO-NAME" list available
```
